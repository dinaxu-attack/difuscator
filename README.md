# difuscator
Difuscator - Simple protector written in Go.


# Install 
```
go install github.com/dinaxu-attack/difuscator 
```

# Usage

You can compile

```
difuscator --file main.go --compile
```

or 

```
difuscator --file main.exe --compile
```
