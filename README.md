# difuscator
Difuscator - Simple protector written in Go.


# Install 
```
go install github.com/dinaxu-attack/difuscator 
```

# Usage

```
difuscator [-o output (Example build.exe) ] [-c compile (bool) ] [-obf (bool) ]
```

# Example 

```
difuscator -o build.exe -c -obf
```
